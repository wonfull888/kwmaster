# kwmaster

`kwmaster` 是一个面向用户结果的关键词工作流 Skill：
它帮助你更快完成关键词收集、整理、优化和落地，减少重复劳动，把时间用在真正能带来增长的决策上。

## 这个 Skill 能帮你什么（先看价值）

- 更快找到高价值关键词：缩短从“想做什么内容”到“可执行关键词清单”的时间
- 降低试错成本：通过结构化流程减少拍脑袋选词
- 提高执行一致性：把关键词研究和优化步骤标准化，避免团队成员各做各的
- 支持持续迭代：让关键词策略可以被复盘、更新和沉淀

## 对用户的直接好处

- 内容团队：更容易规划选题，提升内容命中率
- 增长/SEO 团队：更快定位机会词，提升自然流量效率
- 产品与运营团队：用关键词信号反推用户需求，辅助功能和活动决策
- 个人创作者：减少信息焦虑，用更清晰的方法做内容和分发

## `kwmaster` 解决的核心问题

很多关键词工作卡在三件事上：

1. 信息太散：词表、笔记、分析分散在多个工具
2. 决策太慢：不知道该先做什么词、为什么做
3. 复用太弱：做过的研究难以沉淀成可复用资产

`kwmaster` 的目标就是把这些问题变成可执行、可复盘、可持续优化的流程。

## 适用场景

- 新项目冷启动，需要快速建立关键词策略
- 既有内容增长放缓，需要重新发现机会词
- 多人协作，需要统一关键词研究标准
- 想把“临时分析”升级为“长期方法论”
